# Ajudai (React Native)

Projeto migrado do React Web para React Native.

## Scripts

- `npm start` — inicia o Metro Bundler
- `npm run android` — executa no Android
- `npm run ios` — executa no iOS (MacOS)

## Estrutura
- `App.js`: ponto de entrada
- `src/`: código fonte
- `assets/`: imagens e recursos
